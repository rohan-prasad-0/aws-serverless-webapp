# 🚀 AWS Serverless Contact Form Web App

A fully serverless web application built using AWS cloud services.  
This project demonstrates how a frontend form integrates with backend AWS services using **API Gateway, AWS Lambda, and DynamoDB**.


## 📌 Project Overview

This application allows users to submit their **Name, Email, and Message** through a responsive web form.  
The data is processed through a serverless backend and stored securely in **Amazon DynamoDB**.


## 🏗️ Architecture

Frontend (HTML, CSS, JavaScript)
          ↓
Amazon API Gateway (REST API)
          ↓
AWS Lambda (Backend Logic)
          ↓
Amazon DynamoDB (NoSQL Database)


## ⚙️ Features

- 🌐 Responsive Contact Form UI
- ⚡ Serverless Backend (No servers to manage)
- 🔗 REST API using API Gateway
- 🧠 AWS Lambda function for processing requests
- 💾 Data storage in DynamoDB
- ✅ Input validation (frontend)
- 🔔 Success & error toast notifications
- 📱 Mobile-friendly design


## 🧰 Tech Stack

### Frontend
- HTML5
- CSS3
- JavaScript (Vanilla)

### Backend (AWS)
- Amazon API Gateway
- AWS Lambda
- Amazon DynamoDB


🔄 How It Works
User fills the contact form
Frontend sends POST request to API Gateway
API Gateway triggers AWS Lambda function
Lambda processes and validates data
Data is stored in DynamoDB
Success response is returned to frontend
